# Airline CI/CD Demo

This is a simple Node.js application created to demonstrate a CI/CD pipeline using GitHub Actions.

## Purpose

This demo shows how manual production deployment can be improved using an automated pipeline.

## Current Manual Process

Developer manually copies and pastes files into the production server.

## Proposed CI/CD Process

Developer pushes code to GitHub, and GitHub Actions automatically tests and prepares the application for deployment.

## Technologies Used

- Node.js
- Express.js
- Git
- GitHub
- GitHub Actions
